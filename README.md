## Introduction

Tic Tac Toe is a classic game where two players take turns marking spaces on a 3x3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

## Features

- Play against another player.
- Reset the game at any time.
- Get notified when there's a winner.
- Responsive design, playable on both desktop and mobile devices.

## How to Play

1. Open the index.html file in your web browser.
2. Click on any empty box to place your mark (X or O).
3. The game will automatically switch turns between players.
4. The first player to get three marks in a row (horizontally, vertically, or diagonally) wins the game.
5. Click "New Game" to start a new game at any time.
6. Click "Reset Game" to reset the current game.

## Files Structure

- *index.html*: Contains the structure of the web page.
- *style.css*: Contains the styles for the HTML elements.
- *app.js*: Contains the JavaScript logic for the Tic Tac Toe game.
